# Como executar o projeto

Pré-requisitos: npm / yarn

```bash
# clonar repositório
git clone git@github.com:allpha23/react-portfolio.git

# entrar na pasta do projeto
cd react-portfolio

# instalar dependências
npm install

# executar o projeto
npm run dev
```
